Элементы форм:

  1. form (action, method, target, enctype) - создаёт форму. Данные могут быть обработаны с помощью PHP или JavaScript. Служит контейнером для тегов input, select, button, fieldset. Отправка формы через кнопку input type:submit или через button
  2. input (type [text | password | checkbox | radio | hidden | button | submit | reset | file], value, placeholder, name, disabled, pattern, required) - создает различные элементы формы (поля ввода, checkbox, radio, кнопку). Не требует закрывания.
    Новые в HTML5 [email | number | url | tel | search | color | date | month | week | datetime | datetime-local | range]
  3. checkbox
  4. radio
  5. submit
  6. label (for, accesskey) - задаёт метку для поля ввода input, textarea, select и т.д. Привязывается по id поля ввода. Input может быть вне тега label или внутри его, тогда id указывать не нужно
  7. textarea (name, id, cols, rows, resize, pattern, required) - многострочное поле ввода. Внутри тега можно поместить текст в поле по умолчанию без аттрибута value. Подсказка для ввода с помощью placeholder.
  8. select (multiple, name)- создает выпадающий список. Отдельный пункт списка должен храниться в теге option. Тег option selected выбирает пункт по умолчанию. Multiple позволяет выбрать несколько пунктов меню (select multiple name="items[]").
  9. button (type [button | submit | reset])- создает кнопку
  10. fieldset
    Данные пользователя
    - группирует поля HTML формы рамкой. Подпись рамки с помощью тега legend, поле ввода input
  11. value - может задавать текст, отображаемый внутри поля ввода. Для radio задают разные значения. Для отмеченного чекбокса отсылается значение атрибута, если атрибута нет - отсылается значение "on". Для кнопки отображает надпись на кнопке.
  12. placeholder - подсказка в поле ввода. Пропадает автоматически при попытке ввести текст. Можно применить CSS стили.
  13. ::placeholder
  14. checked
  15. checked
  16. resize (both | horizontal | vertical | none) - запрещает или разрешает изменение размеров textarea. Можно ограничить минимальные и максимальные размеры с помощью max-width, max-width, max-height, max-height
